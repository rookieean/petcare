
# Pet Care

Sebuah aplikasi booking app untuk seseorang yang memiliki hewan peliharaan seperti anjing, kucing, dan hamster, lalu ingin memesan perawatan hewan untuk kesayangan mereka, agar menjaganya tetap sehat dan sentosa ðŸ˜Š

## Features

- Booking hewan peliharaan secara online
- Bebas memilih hari booking antara senin-jumat, 09:00-20-00
- Responsive web
